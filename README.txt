# Vida Pet Inteligente - Página de Links

Arquivos:
- index.html
- style.css
- assets/logo-pet.png

Como editar:
1. Abra o arquivo index.html.
2. Troque todos os textos `COLE_AQUI_SEU_LINK...` pelos seus links reais.
3. Troque a imagem assets/logo-pet.png por uma imagem do Mingau, Caramelo ou logo.
4. Publique no GitHub Pages, Netlify ou Vercel.

Dica:
Mantenha os botões principais no topo:
- Achadinhos da semana
- O Mingau recomenda
- O Caramelo aprovou
